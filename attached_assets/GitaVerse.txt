# Build Shree Bhagwad Gita Wisdom - Bhagavad Gita Verse Web App

## Project Overview
Create a modern, mobile-responsive web application called **Shree Bhagwad Gita Wisdom** that displays random Bhagavad Gita verses in an interactive flashcard format. The app should work seamlessly on both desktop and mobile devices.

## Core Requirements

### Tech Stack
- **Frontend**: React.js with modern hooks
- **Styling**: CSS3 with Flexbox/Grid, mobile-first responsive design
- **Data**: Parse and load from CSV file containing 701 Bhagavad Gita verses
- **Storage**: Browser localStorage for user preferences
- **No backend required** - fully client-side application

### Data Structure (CSV Columns)
The app will use a CSV with these columns:
1. ID (e.g., "BG1.1")
2. Chapter (number)
3. Verse (number) 
4. Shloka (Sanskrit text in Devanagari)
5. Transliteration (Roman script)
6. HinMeaning (Hindi translation)
7. EngMeaning (English translation)
8. WordMeaning (detailed word-by-word explanation - shown on card flip)
